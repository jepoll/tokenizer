# GPT-4 Tokenizer Exercise

Implementation of the first two exercises from minbpe project.

Completed:
- BasicTokenizer
- RegexTokenizer (GPT-4 split pattern)

Features:
- Byte-level BPE training
- Encoding and decoding
- GPT-4 style regex pre-tokenization
- Visualization of learned merges

Reference project:
https://github.com/karpathy/minbpe

The goal of this repository is educational: understanding how modern LLM tokenizers work by implementing them from scratch.
